All around the Web Server

  • A web server is a software program or a hardware device that is responsible for processing client requests for web pages and delivering the requested content to the client
  • HTTP (Hypertext Transfer Protocol) to communicate with the clients
  • Web servers handle HTML pages, JSON, XML, etc. filer formats
