CSS

  • CSS means cascading style sheets
  • is a language used to describe the presentation of a document written in a markup language
  • CSS Fetures are the styling and you can make animations
  • CSS is a living Standard in the web development

    • CSS code can be applied to HTML documents in several ways:

  • Inline styles: CSS styles are applied directly to individual HTML elements using the `style` attribute
  • Embedded styles: CSS styles are defined within the style element in the head section of an HTML document
  • External styles: CSS styles are defined in a separate .css file, which is then linked to an HTML document using the link element in the head section
